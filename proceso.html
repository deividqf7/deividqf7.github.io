Detalle del Proceso: De la Materia Prima al Polímero

  1. Obtención de Materias Primas La mayoría de los plásticos se fabrica a partir de **derivados del petróleo y gas natural**, pero también existen alternativas **más ecológicas** que utilizan materias primas vegetales.
  2. Producción del Monómero (Molécula Base) El primer paso es obtener el **monómero**, una molécula simple que servirá como base para el plástico. Esto se logra mediante procesos químicos, como la **destilación del petróleo**, que separa los distintos componentes según su punto de ebullición. Monómeros comunes incluyen el **etileno**, **propileno** y **estireno**.
  3. Polimerización (Creación del Polímero) En esta etapa, los monómeros se combinan mediante **reacciones químicas** para formar largas cadenas llamadas **polímeros**, que son la verdadera estructura del plástico. Dependiendo del tipo de polimerización (adición o condensación), se obtienen plásticos con características distintas como el **polietileno**, **polipropileno** o **PVC**.

Fase de Moldeo: De Polímero a Producto Útil

  1. Transformación y Moldeo El polímero se calienta y se moldea mediante diferentes técnicas para transformarlo en productos útiles. Las técnicas principales son:
    • **Extrusión:** El plástico fundido se empuja a través de un molde para formar **tubos o láminas**.
    • **Inyección:** Se introduce el material en moldes cerrados para obtener objetos con **formas complejas** como juguetes o piezas de maquinaria.
    • **Soplado:** Técnica utilizada para fabricar **botellas y envases huecos**.
